# MassSafe Test Suite

Comprehensive testing infrastructure for the MassSafe application.

## Test Structure

```
tests/
├── env.test.ts           # Environment variable validation
├── database.test.ts      # Database connection and CRUD operations
├── fbi-api.test.ts       # FBI Crime Data API integration
├── safety-score.test.ts  # Safety score calculation validation
├── search.test.ts        # City search functionality
├── integration.test.ts   # End-to-end workflows
└── setup.ts              # Jest configuration and globals
```

## Running Tests

```bash
# Run all tests
npm test

# Run tests in watch mode
npm run test:watch

# Run tests with coverage report
npm run test:coverage

# Run tests with verbose output
npm run test:verbose

# Run comprehensive verification
npm run verify
```

## Troubleshooting

### Tests Failing

1. Verify all environment variables are set in `.env`
2. Check database connection (run `npm run db:push`)
3. Ensure FBI API key is valid
4. Check internet connectivity for API tests

### Timeout Errors

Increase timeout in `jest.config.js`:

```javascript
testTimeout: 60000; // 60 seconds
```
